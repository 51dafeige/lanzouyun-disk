## 蓝奏云盘

支持暗黑模式，支持 `Windows`, `macOS-x64` 和 `macOS-arm64` 平台。

## 功能

* 主要功能
  * [x] 暗黑模式
  * [x] 文件夹和任意格式文件上传
  * [x] 断点上传 / 下载
  * [x] 文件分割
  * [x] 回收站
  * [x] 链接解析
* 操作（支持文件和文件夹）
  * [x] 多选
  * [x] 编辑
  * [x] 分享
  * [x] 排序
  * [x] 查找
* 批量操作（支持文件和文件夹）
  * [x] 批量上传（支持拖动上传）
  * [x] 批量下载
  * [x] 批量移动
  * [x] 批量删除
  * [x] 批量分享
  * [x] 批量设置密码
